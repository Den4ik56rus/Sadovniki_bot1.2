# Структура папок для документов

Размещайте PDF-файлы в соответствующие папки по категориям и культурам.

## Трёхуровневая система приоритизации:

**УРОВЕНЬ 1 (высший):** Q&A пары из базы знаний (управляется через админку бота)

**УРОВЕНЬ 2 (средний):** Специфичные документы для конкретных типов культур
- `малина_ремонтантная/`, `малина_летняя/`
- `клубника_ремонтантная/` (включает НСД), `клубника_летняя/`

**УРОВЕНЬ 3 (низкий):** Общие документы для всех типов культуры
- `малина общая/`, `клубника общая/`

При ответе пользователю бот использует все уровни с приоритетом 1 > 2 > 3.

## Как добавить документы:

1. Положите PDF-файлы в нужную папку
2. Запустите скрипт импорта:
   ```bash
   source venv/bin/activate
   python scripts/import_documents.py
   ```

## Примеры размещения:

**Общие документы (УРОВЕНЬ 3):**
- `питание_растений/малина общая/удобрения_для_малины.pdf`
- `посадка_и_уход/клубника общая/посадка_клубники.pdf`

**Специфичные документы (УРОВЕНЬ 2):**
- `питание_растений/малина_ремонтантная/удобрения_ремонтантной.pdf`
- `защита_растений/клубника_летняя/вредители_летней_клубники.pdf`

**Общая информация:**
- `питание_растений/общая_информация/основы_питания.pdf`

## Требования к файлам:

- Формат: PDF
- Максимальный размер: 20 МБ
- Кодировка: UTF-8 (предпочтительно)
- Содержание: текст (не только картинки)
