# Структура папок для документов

Размещайте PDF-файлы в соответствующие папки по культурам.

## Двухуровневая система приоритизации:

**УРОВЕНЬ 1 (высший):** Q&A пары из базы знаний (управляется через админку бота)

**УРОВЕНЬ 2 (документы):** PDF-документы по культурам
- Специфичные типы: `малина_ремонтантная/`, `малина_летняя/`, `клубника_ремонтантная/`, `клубника_летняя/`
- Общие: `малина_общая/`, `клубника_общая/`
- Остальные культуры: `голубика/`, `смородина/`, `жимолость/`, `крыжовник/`, `ежевика/`

Приоритет при ответе: Q&A пары (уровень 1) > Документы (уровень 2)

## Структура:

```
data/documents/
├── малина_общая/              # Общие документы для всех типов малины
├── малина_ремонтантная/       # Специфичные для ремонтантной малины
├── малина_летняя/             # Специфичные для летней малины
├── клубника_общая/            # Общие документы для всех типов клубники
├── клубника_ремонтантная/     # Специфичные для ремонтантной/НСД клубники
├── клубника_летняя/           # Специфичные для летней клубники
├── голубика/
├── смородина/
├── жимолость/
├── крыжовник/
├── ежевика/
└── общая_информация/          # Общая информация по всем культурам
```

## Как добавить документы:

1. Положите PDF-файлы в нужную папку (по культуре)
2. Запустите скрипт импорта:
   ```bash
   source venv/bin/activate
   python scripts/import_documents.py
   ```

## Примеры размещения:

**Общие документы:**
- `малина_общая/удобрения_для_малины.pdf`
- `клубника_общая/посадка_клубники.pdf`

**Специфичные документы:**
- `малина_ремонтантная/уход_ремонтантной.pdf`
- `клубника_летняя/вредители_летней_клубники.pdf`

**Общая информация:**
- `общая_информация/основы_питания.pdf`

## Важно:

Документы больше НЕ сортируются по типу вопроса (питание, уход, защита и т.д.).
Сортировка только по культуре растений.

## Требования к файлам:

- Формат: PDF
- Максимальный размер: 20 МБ
- Кодировка: UTF-8 (предпочтительно)
- Содержание: текст (не только картинки)
